About psi-probe

Psi Probe - Advanced manager and monitor for Tomcat, forked from Lambda Probe

Project Modules

This project has declared the following modules:

Name Description
psi-probe-core PSI Probe Core - Core logic, data models, and controllers
psi-probe-rest PSI Probe Rest - RESTful service endpoints.
psi-probe-tomcat85 PSI Probe Tomcat 8.5.x Adapter - Implementation of features specific to Apache Tomcat 8.5.x
psi-probe-tomcat9 PSI Probe Tomcat 9.0.x Adapter - Implementation of features specific to Apache Tomcat 9.0.x
psi-probe-web PSI Probe Web Application - Web view, message resources, and configuration
